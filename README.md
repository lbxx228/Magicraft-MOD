# MagicCraft-MonsterHpRatio
《魔法工艺》BepInEx模组：将所有怪物/精英/BOSS的血量血量×10，移速×2

## 📌 功能介绍
- 对游戏内所有非玩家阵营单位（怪物/精英/BOSS/主动攻击单位）的血量×10
- 适配游戏的DOTS/ECS系统
- 无性能损耗，不影响游戏渲染/稳定性

## 📥 安装要求
- 游戏版本：《魔法工艺》v1.2.5
- 前置依赖：BepInEx 5.4.21+（必须先给游戏安装BepInEx）

## 🛠️ 安装步骤
1. 给《魔法工艺》安装BepInEx：下载BepInEx_x64_5.4.21.0，解压到游戏根目录（MagicCraft.exe同目录），启动游戏一次生成配置文件。
2. 下载本模组的`MonsterHpRatio.zip`,解压后得到`MonsterHpRatio.dll`
3. 将DLL复制到游戏根目录的 `BepInEx/Plugins/` 文件夹。
4. 启动游戏，模组自动生效！

## ✨ 自定义修改（改血量倍率）
如果你想把血量改成3倍/0.5倍，可修改源码中的数值：
1. 打开源码中的 `MonsterHpRatio.cs` 文件。
2. 找到 `cfg.maxHP *= 10f;` 这一行，把`10f`改成你想要的倍率（比如3f=3倍，0.5f=减半）。
3. 重新编译DLL，替换Plugins文件夹中的文件即可。

## ❌ 常见问题
### Q1：模组放进去没生效？
A1：检查3点：
- BepInEx是否安装成功（游戏根目录有BepInEx文件夹）；
- DLL是否放在 `BepInEx/Plugins` 里（不是Core文件夹）；
- 游戏版本是否和模组适配。

### Q2：游戏启动闪退？
A2：大概率是BepInEx版本不匹配，推荐用BepInEx 5.4.21.0版本。
